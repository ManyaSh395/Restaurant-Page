# Restaurant-Page:
A modern single-page restaurant website built using JavaScript, Webpack, and modular architecture.

The project demonstrates dynamic DOM manipulation, tab-based navigation, reusable components, and clean layout design without relying on frameworks.

## Getting Started:
1. Clone the repository:-
git clone git@github.com:ManyaSh395/Restaurant-Page.git
cd restaurant-page

2. Install Dependencies:-
npm install

3. Run Development Server:-
npx webpack serve

Open your browser and go to: http://localhost:8080

## Build for Production:
npx webpack
